Local admin para editar "Matias Elstein.html"

Instrucciones:

1. Abrir una terminal en esta carpeta.
2. Instalar dependencias:

```
npm install
```

3. Iniciar el servidor:

```
npm start
```

4. Abrir en el navegador: http://localhost:3000/admin

Nota: El servidor modifica directamente el archivo `Matias Elstein.html`. Haz una copia si quieres mantener una versión original.
